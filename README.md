# LForM

LForM is a traffic/security(UTM) log viewer for Firewalls(Fortigate 5.4.3).<br>
LForM collects the log of the next-generation firewall, to visualize the traffic and security log.

# Component

- Java : 1.8.0
- ElasticSearch : 6.5.1
- Fluentd(td-agent) : 3.2.1
- kibana : 6.5.1
- nginx : 1.14.1
- Webmin : 1.900
